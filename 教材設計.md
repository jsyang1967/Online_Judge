### 阿姆斯壯數
:::info
所謂 Armstrong number 指的是一個 n 位數的整數，它的所有位數的 n 次方和恰好等於自己。

如:$1634 = 1^4 + 6^4 + 3^4+ 4^4$ 

請依題目需求在一定範圍內找出該範圍內的所有 armstrong numbers.
:::
#### Input
輸入共一行包含兩個數字$n, m(n<m, n>0, m<=1000000)$，代表所有尋找 armstrong number 的範圍
```=
100 999
```
```=
10 99
```

#### 
將所有範圍內的 armstrong number 依序由小到大輸出，如果沒有找到請輸出 none.
```=
153 370 371 407
```
```=
none
```

#### Answer
main.py`

```python=

def spreadNum(num):
    '''enter a string (integer) and returns the split ('3000' -> ['3', '0', '0', '0']) (list)'''
    
    list1 = list(num)
    return list1
    

def calSpread(list1):
    '''input: a list which is splitted (['3', '0', '0']) (list)      
       output: the calculation of the num (integer)'''
    ans = 0
    for i in list1:
        ans = ans + int(i) ** len(list1)
    return ans

while 1:
    try:

        q = input()
    except:
        break

    q = q.split()
    button = int(q[0])
    top = int(q[1])

    ifTrue = False
    for i in range(button, top-1):
        
        if calSpread(spreadNum(str(i))) == i:
            print(i, end=" ")
            ifTrue = True
        
    if ifTrue == False:
            print("none")
    
    print()

```

### 字元處理
:::info
在密碼學簡單的加密方式，就是把明文的每個字元加上某一個整數K而得到密文的字元（明文及密文字元一定都在ASCII碼中可列印的範圍內）。例如若K=2，那麼明文:apple經過加密後就變成密文:crrng了。解密則是反過來做。這個問題是給你一個密碼字串，請你依照上述的加密方式輸出密文。本任務中$K=7$
:::
#### Input
輸入共一行，每行含有1個字串，就是需要加密的明文。
```=
*CDC is the trademark of the Control Data Corporation.
```
```=
*IBM is a trademark of the International Business Machine Corporation.
```
#### Output
對每一測試資料，請輸出加密後的密文。
```=
1JKJ'pz'{ol'{yhklthyr'vm'{ol'Jvu{yvs'Kh{h'Jvywvyh{pvu5
```
```=
1PIT'pz'h'{yhklthyr'vm'{ol'Pu{lyuh{pvuhs'I|zpulzz'Thjopul'Jvywvyh{pvu5
```
#### Answer
main.py`

```python=

trans=[ord(x) for x in input()]
trans=[x+7 for x in trans]
output=[chr(x) for x in trans]
print_="".join(output)
print(print_)

```
